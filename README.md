# Credit Risk Analysis Using Quantum Computers
This repository implements a quantum‑enhanced credit risk analysis workflow inspired by Quantum Risk Analysis (S. Woerner, D. Egger, 2019).
The notebook walks through the circuit step by step and evaluates different approaches for estimating the Value at Risk (VaR), including Maximum Likelihood Amplitude Estimation, Iterative Amplitude Estimation, and Grover‑based techniques.

All code in this repository is original and independently written.
Conceptual foundations are based on the referenced papers.
